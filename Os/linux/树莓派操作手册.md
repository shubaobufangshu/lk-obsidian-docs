# 树莓派操作手册

通过 `uname -m` 确定你使用的系统的架构。

## wifi链接设置

若树莓派无网线连接，可在 boot 分区创建`wpa_supplicant.conf`文件配置 WiFi，系统启动后会自动连接。

在 boot 分区根目录创建`wpa_supplicant.conf`，内容如下（按需修改 WiFi 名、密码和国家代码）：

```txt
country=CN  # 国家代码，中国为CN，美国为US，英国为GB
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1

network={
    ssid="你的WiFi名称"  # 双引号保留，WiFi名含特殊字符也需保留
    psk="你的WiFi密码"   # 双引号保留
    key_mgmt=WPA-PSK     # 加密方式，普通WiFi默认WPA-PSK
    priority=1           # 连接优先级，数字越大越优先
}
```

## ssh链接设置

### 服务端

```bash
# 安装 SSH 服务（CentOS/RHEL）
yum install -y openssh-server
# 安装 SSH 服务（Ubuntu/Debian）
apt install -y openssh-server

# 启动 SSH 服务
systemctl start sshd
# 设置开机自启
systemctl enable sshd
# 查看 SSH 服务状态
systemctl status sshd
```

- 必须安装并启动 **SSH 服务**（sshd），默认端口为 22；
- 开放防火墙 / 安全组的 SSH 端口（如 22），否则无法建立连接；
- 确保用于传输的用户对目标目录有**读写权限**（如上传文件需要目标目录的写入权限）。

树莓派的**boot 分区**是 FAT32 格式的，在 Windows/Linux/macOS 电脑上均可直接识别和编辑，这是实现**无屏幕初始化 SSH 和 VNC**的核心途径。

其中**SSH 可直接通过 boot 分区创建文件开启**，而**VNC 因依赖系统配置和用户权限**，需结合 SSH 先完成基础配置（推荐），或进阶修改 boot/root 分区文件实现。

### 方案一

使用官方烧录工具,在设置中设置ssh和username和password

### 方案二

树莓派官方提供了**无屏幕开启 SSH**的快捷方式：在 boot 分区根目录创建一个**名为`ssh`的空文件（无任何扩展名）**，系统启动时会自动识别并启用 SSH 服务。

- 打开 boot 分区的根目录，右键→新建→文本文档。
- 将文本文档的名称改为`ssh`（**删除`.txt`扩展名**，需先在「文件资源管理器」→「查看」中勾选「文件扩展名」）。
- 双击打开该文件，删除所有内容（保持为空），保存后关闭。

## VNC

1. 打开终端（或通过 SSH 连接树莓派），输入以下命令进入配置界面：
   ```bash
   sudo raspi-config
   ```
2. 用方向键选择 **3 Interface Options**（接口选项），按回车。

3. 选择 **I3 VNC**，按回车。

4. 提示 “Would you like the VNC Server to be enabled?” 时，选择 **Yes**，按回车。

5. 确认 VNC 已启用后，按 **ESC** 键退出配置界面，系统会自动重启 VNC 服务。

其他方法:电脑自带远程桌面链接

- ```
  sudo apt-get install xrdp
  ```

- 电脑自带远程桌面链接,填写ip地址

## 中文设置

1. 打开配置工具：

   ```bash
   sudo raspi-config
   ```

2. 依次选择：

   ```
   5 Localisation Options
   ```

   →

   ```
   L1 Locale
   ```

3. 在列表中找到并勾选：
   - ```
     zh_CN.UTF-8 UTF-8
     ```

     （中文 UTF-8 编码，按空格键勾选）

     （可保留默认的

     ```
     en_GB.UTF-8
     ```

     作为备选，避免部分软件兼容问题）

4. 按 `Tab` 键切换到 `OK` 并回车，选择默认语言时，选 `zh_CN.UTF-8`。
5. 完成后按 `ESC` 退出配置工具，提示重启时选择 `Yes`。

## raspi-config 工具

`raspi-config` 是树莓派官方提供的命令行配置工具

- 打开方式`sudo raspi-config`
  - ```bash
    sudo raspi-config
    ```

- 主要功能选项（通过上下键导航，回车确认）：
  - **System Options**：系统设置（主机名、密码、时区、键盘布局等）。
  - **Display Options**：显示设置（分辨率、 Overscan 等）。
  - **Interface Options**：接口设置（启用 SSH、VNC、SPI、I2C 等）。
  - **Performance Options**：性能设置（超频、内存分配等）。
  - **Localisation Options**：本地化设置（语言、区域、WiFi 国家码等）。
  - **Advanced Options**：高级设置（扩展文件系统、设置环境变量等）。

### 核心工具

安装完成后，原来的相机工具会以 `rpicam-*` 命令形式存在，例如：

#### 拍照

- `rpicam-still`（拍照）
  - 拍摄照片：`rpicam-still`（替代原来的 `libcamera-still`）
    - ```bash
      # 立即拍照，保存为 photo.jpg（默认分辨率）
      rpicam-still -o photo.jpg
      ```

    - ```
      # 延时5秒拍照（-t 单位：毫秒）
      rpicam-still -o delay_photo.jpg -t 5000
      ```

    - ```bash
      # 自定义分辨率（宽800，高600）
      rpicam-still -o custom_size.jpg -w 800 -h 600
      ```
    - ```bash
      # 4K分辨率（3840x2160）、95质量、自动对焦，保存为 HQ_photo.jpg
      rpicam-still -o HQ_photo.jpg -w 3840 -h 2160 --quality 95 --autofocus --nopreview
      ```

##### 常用拍照参数

| 参数           | 说明                                       | 示例                      |
| -------------- | ------------------------------------------ | ------------------------- |
| `-o/--output`  | 指定输出文件路径（支持 jpg/png/bmp）       | `-o ~/Pictures/photo.png` |
| `-t/--timeout` | 拍照延时（毫秒），0 表示无限等待           | `-t 3000`                 |
| `-w/--width`   | 图像宽度                                   | `-w 1920`                 |
| `-h/--height`  | 图像高度                                   | `-h 1080`                 |
| `--nopreview`  | 禁用实时预览（无屏场景）                   | `--nopreview`             |
| `--quality`    | 图像质量（1-100，默认 85）                 | `--quality 95`            |
| `--autofocus`  | 启用自动对焦（仅 Camera 3/HQ Camera 支持） | `--autofocus`             |
| `--focus`      | 手动对焦（单位：米，0 表示无限远）         | `--focus 0.5`             |

#### 录像

- `rpicam-vid`（录像）
  - 录制视频：`rpicam-vid`（替代原来的 `libcamera-vid`）
    - ```bash
      # 录制10秒视频，保存为 video.h264（默认分辨率、帧率30）
      rpicam-vid -o video.h264 -t 10000
      ```

    - ```
      # 录制MP4格式视频（需配合 ffmpeg 转码，下文会讲）
      rpicam-vid -o video.h264 -t 10000 && ffmpeg -i video.h264 -c copy video.mp4
      ```

##### 常用录像参数

| 参数                       | 说明                               | 示例                      |
| -------------------------- | ---------------------------------- | ------------------------- |
| `-o/--output`              | 指定输出文件（默认 H264 格式）     | `-o ~/Videos/record.h264` |
| `-t/--timeout`             | 录像时长（毫秒），0 表示无限录制   | `-t 20000`                |
| `-w/--width`/`-h/--height` | 视频分辨率                         | `-w 1920 -h 1080`         |
| `--framerate/-fps`         | 帧率（fps）                        | `--framerate 60`          |
| `--codec`                  | 编码格式（h264/h265）              | `--codec h265`            |
| `--bitrate`                | 视频码率（单位：Kbps，默认 10000） | `--bitrate 20000`         |
| `--nopreview`              | 禁用预览                           | `--nopreview`             |
| `--autofocus`              | 启用自动对焦                       | `--autofocus`             |

#### rpicam-jpeg

要捕捉全分辨率的JPEG图像并保存到名为 的文件，请执行以下命令：test.jpg

```bash
rpicam-jpeg --output test.jpg
```

使用超时选项来更改预览窗口的显示时间。宽度和高度选项会改变已保存图像的分辨率。例如，以下命令显示预览窗口2秒，然后捕获并保存分辨率为640×480像素的图像：

```bash
rpicam-jpeg --output test.jpg --timeout 2000 --width 640 --height 480
```

### 高级配置

#### 基础配置

```bash
# 手动设置曝光时间（单位：微秒，1000微秒=1毫秒）
rpicam-still -o exp_photo.jpg --shutter 5000

# 设置ISO感光度（100-800，数值越高越亮）
rpicam-still -o iso_photo.jpg --iso 400

# 自定义白平衡（红/蓝增益，默认1.0）

```

#### 网络流媒体推流

通过 `rpicam-jpeg` 或 `rpicam-vid` 配合 `ffmpeg`，可将相机画面推流到局域网 / 互联网（以 RTSP 推流为例）：

1. #### RTSP 服务器

```bash
# 下载并启动 rtsp-simple-server
wget https://github.com/aler9/rtsp-simple-server/releases/download/v0.25.0/rtsp-simple-server_linux_arm64.tar.gz
tar -xzf rtsp-simple-server_linux_arm64.tar.gz
./rtsp-simple-server &  # 后台运行
```

2. #### 推流到 RTSP 服务器

```bash
# 1080P 30帧推流到 rtsp://localhost:8554/stream
rpicam-vid -w 1920 -h 1080 --framerate 30 --nopreview -o - | ffmpeg -i - -c:v copy -f rtsp rtsp://localhost:8554/stream
```

3. #### 播放流

在电脑 / 手机上使用 VLC 播放器，打开 `rtsp://树莓派IP:8554/stream` 即可实时观看

## 文件传输

#### SCP 命令

适用于 Linux/macOS 终端或 Windows PowerShell

- **SCP**：主打**一次性快速传输**，语法简单，适合单文件 / 目录的直接拷贝；

- ```
  scp [可选参数] 源文件/目录 [用户@]主机地址[:端口]:文件/目录路径
  ```

| 参数 | 说明                                                               |
| ---- | ------------------------------------------------------------------ |
| `-P` | 指定远程 SSH 端口（默认 22，**注意是大写 P**，小写 -p 是保留属性） |
| `-r` | 递归传输目录（传输文件夹时必须加此参数）                           |
| `-p` | 保留文件的权限、修改时间、访问时间等属性                           |
| `-v` | 显示详细的传输过程（排错时常用）                                   |
| `-q` | 静默模式，不显示传输进度和提示                                     |
| `-l` | 限制传输速率（单位：Kbit/s，如 `-l 1024` 限制为 1MB/s）            |
| `-C` | 启用压缩传输，减少网络带宽占用（适合大文件）                       |

- 从电脑上传文件到树莓派：

  ```bash
  # 格式：scp 本地文件路径 用户名@树莓派IP:目标路径
  scp ~/Desktop/test.txt pi@192.168.1.100:/home/pi/Documents/
  ```

* 从树莓派下载文件到电脑：

  ```bash
  # 格式：scp 用户名@树莓派IP:文件路径 本地目标路径
  scp pi@192.168.1.100:/home/pi/photo.jpg ~/Pictures/
  ```

#### SFTP 工具

SFTP 是交互式工具，连接远程主机后会进入专属的操作终端，支持目录导航、文件上传 / 下载、创建 / 删除目录等操作。

```bash
sftp [可选参数] [用户@]主机地址[:端口]
```

SFTP 常用交互式命令

SFTP 命令分为**本地操作**和**远程操作**，本地命令前通常加 `l`（local）前缀，核心命令如下：

| 命令                         | 说明                        | 示例                                                                                 |
| ---------------------------- | --------------------------- | ------------------------------------------------------------------------------------ |
| **导航与查看**               |                             |                                                                                      |
| `cd 路径`                    | 切换远程主机的目录          | `cd /opt`                                                                            |
| `lcd 路径`                   | 切换本地主机的目录          | `lcd /home/local`                                                                    |
| `pwd`                        | 查看远程主机当前目录        | `pwd`                                                                                |
| `lpwd`                       | 查看本地主机当前目录        | `lpwd`                                                                               |
| `ls`                         | 列出远程主机当前目录的文件  | `ls` / `ls -l`（详细信息）                                                           |
| `lls`                        | 列出本地主机当前目录的文件  | `lls` / `lls -l`                                                                     |
| **文件传输**                 |                             |                                                                                      |
| `get 远程文件 [本地路径]`    | 下载远程文件到本地          | `get remote.txt`（下载到本地当前目录）`get remote.txt /home/local/new.txt`（重命名） |
| `get -r 远程目录 [本地路径]` | 递归下载远程目录到本地      | `get -r /opt/data /home/local/`                                                      |
| `put 本地文件 [远程路径]`    | 上传本地文件到远程          | `put local.txt`（上传到远程当前目录）`put local.txt /opt/new.txt`（重命名）          |
| `put -r 本地目录 [远程路径]` | 递归上传本地目录到远程      | `put -r /home/local/data /opt/`                                                      |
| `mget 通配符`                | 批量下载远程文件（支持 \*） | `mget *.txt`（下载所有 txt 文件）                                                    |
| `mput 通配符`                | 批量上传本地文件（支持 \*） | `mput *.log`（上传所有 log 文件）                                                    |
| **文件 / 目录管理**          |                             |                                                                                      |
| `mkdir 目录名`               | 在远程主机创建目录          | `mkdir new_dir`                                                                      |
| `lmkdir 目录名`              | 在本地主机创建目录          | `lmkdir local_dir`                                                                   |
| `rm 文件名`                  | 删除远程主机的文件          | `rm useless.txt`                                                                     |
| `rmdir 目录名`               | 删除远程主机的空目录        | `rmdir empty_dir`                                                                    |
| `rename 原文件名 新文件名`   | 重命名远程文件              | `rename old.txt new.txt`                                                             |
| `chmod 权限 文件名`          | 修改远程文件权限            | `chmod 644 test.txt`                                                                 |
| **其他操作**                 |                             |                                                                                      |
| `binary`                     | 以二进制模式传输（默认）    | `binary`                                                                             |
| `ascii`                      | 以文本模式传输              | `ascii`                                                                              |
| `!命令`                      | 执行本地主机的 Shell 命令   | `!ls`（查看本地文件）、`!pwd`（查看本地路径）                                        |
| `exit/quit/bye`              | 退出 SFTP 环境              | `exit`                                                                               |

图形化操作

- **SFTP**：提供**交互式文件操作环境**，支持目录导航、批量操作、文件编辑等，功能更丰富，适合复杂的文件管理。

- Windows：使用 FileZilla，输入树莓派 IP、用户名、密码，端口默认 22，连接后可拖拽文件。

## 防火墙配置（UFW）

树莓派默认无防火墙限制，如需开启：

1. 安装 UFW：

   ```bash
   sudo apt install -y ufw
   ```

2. 基本配置：

   ```bash
   sudo ufw allow ssh  # 允许SSH连接
   sudo ufw allow 5900  # 允许VNC连接（默认端口）
   sudo ufw enable  # 启用防火墙
   sudo ufw status  # 查看防火墙状态
   ```

## 蓝牙设置（连接设备）

1. 打开蓝牙工具：

   ```bash
   bluetoothctl
   ```

2. 进入交互模式后操作：

   ```bash
   power on  # 开启蓝牙
   discoverable on  # 设为可发现
   scan on  # 扫描附近设备（找到设备MAC地址，如XX:XX:XX:XX:XX:XX）
   pair XX:XX:XX:XX:XX:XX  # 配对设备（按提示输入配对码）
   connect XX:XX:XX:XX:XX:XX  # 连接设备
   exit  # 退出交互模式
   ```

##### 安装和卸载包

```
# 安装包
conda install 包名
conda install numpy pandas matplotlib

# 查看当前电脑上所有已创建的 Conda 虚拟环境
conda env list
conda info --envs

# 安装指定版本
conda install python=3.8
conda install numpy=1.21

# 从特定channel安装
conda install -c conda-forge 包名

# 卸载包
conda remove 包名

# 更新包
conda update 包名
conda update --all  # 更新所有包
```

##### 查看包信息

```
# 查看已安装的包
conda list

# 查看特定环境的包
conda list -n 环境名

# 搜索包
conda search 包名

# 查看包信息
conda info 包名
```

#### 环境导出和导入

```
# 导出环境配置
conda env export > environment.yml

# 从文件创建环境
conda env create -f environment.yml

# 导出精简版环境配置（只包含明确安装的包）
conda env export --no-builds > environment.yml

# 更新环境配置
conda env update -f environment.yml
```

#### 配置管理

```
# 查看conda配置
conda config --show

# 添加channel
conda config --add channels 频道名

# 设置channel优先级
conda config --set channel_priority strict

# 移除channel
conda config --remove channels 频道名
```

#### 实用命令

```
# 查看conda信息
conda info

# 清理缓存
conda clean --all

# 检查conda更新
conda update conda

# 查看conda版本
conda --version
```

#### Jupyter相关

```
# 安装jupyter
conda install jupyter

# 在特定环境中安装ipykernel
conda install ipykernel

# 将环境添加到jupyter
python -m ipykernel install --user --name 环境名 --display-name "显示名称"

# 启动jupyter
jupyter notebook
jupyter lab
```

#### 常用技巧

1. **批量安装包**：

```
conda install numpy pandas matplotlib scikit-learn jupyter
```

1. **创建特定用途环境**：

```
# 数据科学环境
conda create -n datascience python=3.9 pandas numpy matplotlib seaborn scikit-learn jupyter

# Web开发环境
conda create -n webdev python=3.9 django flask requests beautifulsoup4
```

1. **查看环境差异**：

```
conda compare environment.yml
```

# 没有验证的内容

boot分区创建用户和密码

树莓派 OS **Bookworm 及以后版本** 移除了默认的`pi`用户，需在 boot 分区创建`userconf.txt`文件配置用户名和加密密码，否则无法登录系统（旧版 Bullseye 可跳过此步）。

### 步骤 1：生成 SHA-512 加密的密码

密码需通过**OpenSSL**生成加密串（Windows 需安装[Git Bash](https://git-scm.com/)或 OpenSSL，Linux/macOS 自带）：

1. 打开终端（Linux/macOS）或 Git Bash（Windows）。
2. 执行命令生成加密密码：`openssl passwd -6`（`-6`表示使用 SHA-512 加密，是树莓派推荐的方式）。
3. 按提示输入自定义密码（如`raspberry123`），再次确认后，终端会输出一串加密字符串（如`$6$xxxxxx$xxxxxxxxxxxxxxxxxxxxxxxxxxxx`）。

### 步骤 2：创建`userconf.txt`文件

在 boot 分区根目录创建名为`userconf.txt`的文件，内容格式为：**用户名：加密密码**，示例：

```txt
pi:$6$eGpZ7L0s$9kXk2U5Y7H1s8D3f6G9j2K7N8B5V3C1A9S7D2F4G6H8J0K2L4M6N8P0Q2R4T6Y8U0I2O4P6A8S9D7F5G3H1J9K7L5M3N1B7V5C3X1Z9E7R5T3Y1U9I7O5P3A1S7D5F3G1H9J7K5L3M1N9B7V5C3X1`
```
