## 初始化
### 初始化设置
#### 激活windows，office
激活windows是使用windows的基础，也是后续对windows进行设置的基础
##### 方式一
+ 键入可移动磁盘，加载激活工具光驱
+ 进入光驱内部，点击激活工具
+ 选择激活方式，选择激活内容
+ 点击激活
使用这类工具进行激活属于侵犯软件著作权的盗版行为，同时违反了法律法规、软件使用协议以及伦理规范，这类行为不仅会损害软件开发者的合法权益，还可能给你的设备带来恶意程序、系统不稳定、数据泄露等安全风险，因此我不建议你使用这一种方式。
##### 方式二
+ 右击windows图标
+ 选择设置
+ 选择激活
+ 选择产品秘钥
+ 输入产品秘钥
产品秘钥获取需要重正规渠道获取

##### 启用存储感知
设置 → 系统 → 存储 → 开启存储感知，配置每周自动清理
#####  虚拟内存优化
+ 右键此电脑 → 属性 → 高级系统设置 → 高级 → 性能设置 → 高级 → 虚拟内存更改
+ 取消 "自动管理所有驱动器的分页文件大小"
+ 推荐设置：
  - 系统管理的大小（默认，适合大多数用户）
  - 手动设置：初始大小 = 内存容量 ×1.5，最大值 = 内存容量 ×2（如 16GB 内存设为 24GB/32GB）
+ 将页面文件移至非系统盘（SSD 最佳），提升读写性能
##### 磁盘设置
###### 分区设置
分区既是为了使**系统与数据分离**又是为了**分类管理数据**
分区数量约3-4个为最佳
C盘容量：200-400最佳，存放系统文件
D盘：存放数据文件
E盘：存放软件本体
+ 右击windows图标，选择磁盘管理
+ 选择要分的磁盘，右键新建或者压缩卷
+ 取决于你是要从c盘里面要空间还是重新加载一部分空间
###### NTFS 优化
关闭磁盘碎片整理计划（SSD）：优化驱动器 → 选择 SSD → 更改设置 → 取消计划运行
##### BIOS/UEFI 优化（不建议轻易开启）
开启 XMP/EXPO（内存超频），提升内存性能 30%+
关闭安全启动（如安装特定软件 / 系统），开启快速启动（UEFI 模式）

#### 浏览器设置
##### 基础
为什么会要讲这个呢，因为windows里面的浏览器几乎达到了系统一样的地位。功能复杂，资源消耗多，而且windows的系统集成度简直离谱，以至于有时候能在日常里面恶心人
- 禁用「个性化广告」：设置 → 隐私、搜索和服务 → 广告和体验个性化 → 关闭所有选项
- 清除浏览数据快捷键：`Ctrl+Shift+Del`，建议每周清理（勾选历史记录、Cookie、缓存）
设置 → 系统和性能

| 设置项     | 推荐配置               | 适用场景                                   |
| ---------- | ---------------------- | ------------------------------------------ |
| 启动增强   | 关闭                   | 减少后台常驻，节省内存（SSD 用户影响小）   |
| 后台运行   | 完全禁用               | 关闭「关闭 Edge 后继续运行后台扩展和应用」 |
| 效率模式   | 开启                   | 自动优化资源，电量低时更积极               |
| 睡眠标签页 | 5 分钟无操作后休眠     | 大幅减少内存占用（20 + 标签时效果显著）    |
| 硬件加速   | 开启（兼容问题时关闭） | 利用 GPU 加速渲染，提升流畅度              |
| 内存节省   | 始终启用               | 自动压缩不活动标签内存，减少 30%+ 占用     |

- 禁用「预加载新标签页」：设置 → 新标签页 → 关闭「预加载新标签页内容」，减少启动资源占用
- 睡眠标签页快捷键：`Ctrl+Shift+P`（手动休眠不常用标签）
- 垂直标签页：启用后节省水平空间 → 点击标签页左侧「垂直标签页」图标（推荐宽屏显示器）
- 关闭「标签页预览」：设置 → 外观 → 关闭「显示标签页预览」，减少内存占用
##### 进阶
###### 启动与新标签页优化（提升启动速度）
1. 启动设置：设置 → 启动、主页和新标签页 → 选择「打开新标签页」或「特定页面」（避免恢复上次会话）
2. 新标签页定制：
   - 关闭「Microsoft Start 信息流」（默认页面消耗资源）
   - 自定义背景：选择纯色或个人图片，禁用动态内容
   - 移除不需要的快捷方式：右键快捷方式 → 移除
######  外观与布局优化（视觉与效率平衡）
1. 主题与模式：设置 → 外观 → 选择「深色模式」（护眼 + 省电），或自定义颜色
2. 工具栏精简：
   - 右键工具栏 → 取消勾选「收藏夹栏」「Copilot 按钮」「侧边栏」等非必要组件
   - 隐藏「必应聊天」图标：设置 → 外观 → 关闭「显示 Copilot 按钮」
3. 字体优化：设置 → 外观 → 自定义字体 → 选择清晰易读的字体（如 Segoe UI），大小 120%（护眼）
###### 侧边栏管理（提升效率或彻底禁用）
   1. 禁用侧边栏（推荐，减少干扰）：右键侧边栏 → 关闭侧边栏，或设置 → 侧边栏 → 关闭「始终显示侧边栏」
   2. 保留侧边栏优化（高效用户）：
      - 仅保留「收藏夹」「历史记录」等必要功能
      - 关闭「自动打开 Copilot 侧边栏」和「推荐内容」
##### 进阶
###### 高级隐私与安全配置
禁用遥测数据：设置 → 隐私、搜索和服务 → 诊断数据 → 设为「基本」，关闭「发送可选诊断数据」
阻止自动安装扩展：设置 → 扩展 → 关闭「自动安装来自 Microsoft 的扩展」
安全浏览模式：右键 Edge 图标 → 新建 InPrivate 窗口（无痕模式），适合敏感操作
###### 浏览器修复与重置（解决顽固问题）
修复 Edge：设置 → 关于 Microsoft Edge → 修复 Microsoft Edge（保留数据）
重置 Edge：设置 → 重置设置 → 恢复默认设置（清除扩展和自定义，保留书签和密码）
彻底清理：
关闭所有 Edge 窗口 → 任务管理器结束所有 msedge 进程
删除用户数据：%localappdata%\Microsoft\Edge\User Data\Default（谨慎操作，备份重要数据）
