Calendar是Obsidian生态中最强大的**时间管理与日记入口插件**（作者Liam Cain），它不仅能替代核心日记插件，还能在日历视图上直观展示每日的字数、任务、链接、反向链接等数据，完美联动Tasks、Templater、Dataview等核心插件，打造一体化的个人知识与时间管理体系。
## 核心配置详解
### General Settings（通用设置）
这是最基础的全局设置，对应你截图的最上方部分：

| 选项 | 作用 | 推荐配置 |
|------|------|----------|
| Confirm before creating new note | 点击空白日期创建新日记前弹出确认框 | ✅ 开启（防止误点创建大量空笔记） |
| Show week numbers | 在日历左侧添加周数列，显示每年的第几周 | 按需开启（适合做周复盘的用户） |
### Sources（数据源配置）⭐ 核心特色
这是Calendar插件区别于其他日历的核心功能，它能在日历日期上显示不同类型的统计数据，每个数据源对应一个颜色圆点和数字。你截图中已经开启了前5个，下面逐一说明：
- **Words（黄色）**：统计当天日记的总字数，直观反映每日的写作量
- **Tasks（橙色）**：统计当天笔记中包含的任务总数（与Tasks插件深度联动）
- **Links（绿色）**：统计当天日记中添加的正向链接数量
- **Backlinks（蓝色）**：统计有多少其他笔记链接到了当天的日记
- **Zettels（紫色）**：统计当天创建的**非日记类普通笔记**数量
- **Emoji Tags（灰色）**：显示当天日记中使用的emoji标签（如📅、✅、⚠️），适合快速标记当日状态

### Localization（本地化设置）
控制日历的语言和日期习惯，你的配置已经非常适合中国用户：

- **Start week on**：一周的开始日期
  - 你的配置是`Locale default (Monday)`（周一），符合中国习惯，无需修改
  - 如需周日开始可手动选择`Sunday`
- **Locale**：区域语言设置
  - 你的配置是`Same as system (zh-cn)`，会自动显示中文的月份和星期名称，保持默认即可
### Notes（日记设置）⭐ 最重要的配置
这部分是你截图中没有显示的，但直接决定了日记的创建规则，**必须正确配置**：

| 选项 | 作用 | 推荐配置 |
|------|------|----------|
| Date format | 日记文件名的日期格式 | `YYYY-MM-DD`（标准格式，所有Obsidian插件都兼容，绝对不要改成中文格式如"2026年05月16日"） |
| Note location | 日记文件的存放文件夹 | 建议设置为专门的文件夹，如`日记/`（避免根目录混乱） |
| Template file location | 日记模板文件的路径 | 如`模板/每日日记模板.md`（与Templater插件联动，自动套用模板） |
| Automatically create note when clicking on a date | 点击日期时自动创建笔记 | ❌ 关闭（配合上面的确认框使用，更安全） |
| Open note in new tab | 在新标签页打开日记 | ✅ 开启（不会覆盖当前正在编辑的笔记） |
| Show today's note in sidebar | 在日历侧边栏显示今日笔记摘要 | ✅ 开启 |
### Week Numbers（周数设置）
只有开启了"Show week numbers"才会显示这个模块：
- **Week number calculation**：周数计算标准
  - 推荐选择`ISO 8601`（国际标准，周一为一周开始，第一周包含本年至少4天），与中国的周数习惯一致
### Hover Preview（悬停预览）
- **Enable hover preview**：鼠标悬停在日期上时显示当日笔记的预览内容
  - ✅ 强烈推荐开启，不用点击就能快速查看当日内容
- **Preview size**：调整预览窗口的大小，根据你的屏幕尺寸调整即可
### Hotkeys（快捷键设置）
为常用操作设置快捷键，大幅提升效率，推荐设置：
- 打开/关闭日历视图：`Ctrl+Shift+C`
- 快速创建今日日记：`Ctrl+Shift+T`
- 跳转到今天：`Ctrl+Shift+H`
- 跳转到上一天：`Ctrl+Shift+Left`
- 跳转到下一天：`Ctrl+Shift+Right`
### Advanced（高级设置）
- **Compact mode**：紧凑模式，减少日历单元格的间距，适合小屏幕用户
- **Show month/year navigation buttons**：显示月份和年份的快速导航按钮
- **Use relative dates**：显示相对日期（如"今天"、"昨天"）而不是绝对日期
- **Custom colors**：自定义所有数据源和日历元素的颜色，打造个性化界面
## 基础使用方法
- **查看日历**：点击左侧边栏的日历图标，或使用快捷键打开
- **创建日记**：点击任意空白日期，在弹出的确认框中点击"Create"，即可自动创建对应日期的日记并套用模板
- **查看当日内容**：鼠标悬停在日期上查看预览，点击日期打开完整日记
- **导航切换**：点击日历顶部的左右箭头切换月份，点击年份可以快速跳转到任意年份，点击"Today"按钮一键回到今天
