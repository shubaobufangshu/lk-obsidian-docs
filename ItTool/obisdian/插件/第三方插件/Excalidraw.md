
## 一、基础安装
1. 关闭安全模式，社区插件搜：**Excalidraw** 安装启用
2. 自动生成绘图后缀 `.excalidraw`，本地存储不联网

## 二、快速新建
- `Ctrl+P` → 输入 **New Excalidraw Drawing** 新建画布
- 右键文件夹 → 新建 Excalidraw 绘图

## 三、核心快捷键（必记）
| 快捷键                | 功能             |
| ------------------ | -------------- |
| `V`                | 选择工具（默认）       |
| `R`                | 矩形             |
| `O`                | 圆形/椭圆          |
| `L`                | 直线             |
| `A`                | 箭头             |
| `T`                | 文字             |
| `E`                | 手绘自由画笔         |
| `Ctrl+Z` 撤销        | 通用撤回           |
| `Ctrl+S` 保存        | 自动保存可不用        |
| `Delete`           | 删除选中元素         |
| `Ctrl+D`           | 快速复制元素         |
| **Ctrl + 方向键（上下左右） | 自动生成子节点 + 自动连线 |

## 四、Obsidian最强联动（精髓）
1. **画布内跳转笔记**
直接输入 `[[笔记名]]`，自动变成链接方块，点击直达笔记
2. **嵌入画布到普通笔记**
```md
![[画布名字.excalidraw]]
```
3. **笔记反向链接画布**
画布和笔记互相生成双链，构建知识网

## 五、实用功能
1. **手绘风格切换**
顶部菜单关闭「手绘风格」变标准商务线条图
2. **插入图片**
直接拖拽图片进画布，自由缩放摆放
3. **导出**
右上角 → 导出 PNG/SVG/PDF 发文档、PPT
4. **无限画布**
鼠标滚轮平移，无边界随便画

## 六、最常用绘图场景
1. 学习：知识点思维导图、背诵框架
2. 编程：代码流程、项目架构、逻辑图
3. 日常：计划排版、思路梳理、草稿草图
4. 工作：方案流程图、分工图

## 七、新手极简设置
1. 默认画布尺寸调大
2. 开启**自动吸附对齐**，连线更整齐
3. 字体调成中文清晰模式

## 八、小技巧
- 按住**空格**临时拖动整张画布
- 框选多个元素可统一移动、改颜色
- 箭头对准方块自动吸附连接
