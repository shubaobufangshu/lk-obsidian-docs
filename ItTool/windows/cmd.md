

## 1. 概述

CMD（命令提示符）是Windows系统内置的命令行解释器，用于执行各类系统操作命令，支持批处理脚本、磁盘管理、权限配置、文件操作等核心场景。本文档整理了CMD核心命令的语法、参数、使用说明及示例，按功能分类结构化呈现，便于查阅和使用。

## 2. 核心命令分类详解

### 2.1 命令解释器基础（CMD）

#### 语法

```batch
CMD [/A | /U] [/Q] [/D] [/E:ON | /E:OFF] [/F:ON | /F:OFF] [/V:ON | /V:OFF] [[/S] [/C | /K] string]
```

#### 参数说明

| 参数     | 核心说明                              |
| ------ | --------------------------------- |
| /C     | 执行指定命令后终止CMD窗口                    |
| /K     | 执行指定命令后保留CMD窗口                    |
| /S     | 修改/C、/K后字符串的引号处理逻辑                |
| /Q     | 关闭命令回显                            |
| /D     | 禁止从注册表执行AutoRun自动命令               |
| /A     | 内部命令输出（管道/文件）使用ANSI编码             |
| /U     | 内部命令输出（管道/文件）使用Unicode编码          |
| /T:fg  | 设置控制台前景/背景颜色（fg为两位十六进制值）          |
| /E:ON  | 启用命令扩展（默认启用）                      |
| /E:OFF | 禁用命令扩展                            |
| /F:ON  | 启用文件/目录名自动补全（Ctrl+D=目录、Ctrl+F=文件） |
| /F:OFF | 禁用文件/目录名自动补全                      |
| /V:ON  | 启用延迟环境变量扩展（通过`!var!`引用变量）         |
| /V:OFF | 禁用延迟环境变量扩展                        |

#### 关键特性说明

- **命令扩展**：默认启用，可通过注册表（`HKEY_LOCAL_MACHINE\Software\Microsoft\Command Processor\EnableExtensions`）或`SETLOCAL ENABLEEXTENSIONS`控制，影响DEL、CD、FOR等命令的功能增强；
- **延迟变量扩展**：默认禁用，启用后可在FOR循环、批处理中实时扩展变量（区别于`%var%`的静态扩展）；
- **AutoRun**：CMD启动时自动执行注册表`AutoRun`项（`HKLM/HKCU\Software\Microsoft\Command Processor\AutoRun`）的命令，`/D`可禁用此行为。

### 2.2 文件/目录权限管理

#### 2.2.1 CACLS（不推荐，建议使用ICACLS）

##### 语法

```batch
CACLS filename [/T] [/M] [/L] [/S[:SDDL]] [/E] [/C] [/G user:perm] [/R user [...]] [/P user:perm [...]] [/D user [...]]
```

##### 核心参数

| 参数         | 说明                                                 |
| ------------ | ---------------------------------------------------- |
| filename     | 目标文件/目录路径                                    |
| /T           | 递归处理当前目录及所有子目录的文件                   |
| /E           | 编辑ACL（访问控制列表）而非覆盖                      |
| /G user:perm | 授予用户权限（perm：R=读、W=写、C=更改、F=完全控制） |
| /P user:perm | 替换用户权限（perm：N=无、R/W/C/F）                  |
| /D user      | 拒绝指定用户访问                                     |

#### 2.2.2 ICACLS（推荐，权限管理增强版）

##### 语法

```batch
:: 保存/恢复ACL
ICACLS name /save aclfile [/T] [/C] [/L] [/Q]
ICACLS directory /restore aclfile [/C] [/L] [/Q]

:: 所有者/ACL检查
ICACLS name /setowner user [/T] [/C] [/L] [/Q]
ICACLS name /findsid Sid [/T] [/C] [/L] [/Q]
ICACLS name /verify [/T] [/C] [/L] [/Q]
ICACLS name /reset [/T] [/C] [/L] [/Q]

:: 权限授予/拒绝/删除
ICACLS name [/grant[:r] Sid:perm[...]] [/deny Sid:perm [...]] [/remove[:g|:d]] Sid[...]] [/T] [/C] [/L] [/Q] [/setintegritylevel Level:policy[...]]
```

##### 参数说明

| 子命令              | 核心作用                                        |
| ------------------- | ----------------------------------------------- | --------------------------------------------- | -------------------------------------------------- |
| /save aclfile       | 将文件/目录的DACL（自主访问控制列表）保存到文件 |
| /restore aclfile    | 从文件恢复DACL到目标目录                        |
| /setowner user      | 修改文件/目录的所有者                           |
| /grant[:r] Sid:perm | 授予权限（`:r`替换原有显式权限，无则追加）      |
| /deny Sid:perm      | 显式拒绝指定权限（优先于授予权限）              |
| /remove[:g          | :d] Sid                                         | 删除SID相关权限（`:g`删除授予、`:d`删除拒绝） |
| /setintegritylevel  | 设置完整性级别（L=低、M=中、H=高）              |
| /inheritance:e      | d                                               | r                                             | 控制继承（e=启用、d=禁用并复制ACE、r=删除继承ACE） |

##### 权限标识对照表

| 简单标识 | 说明       | 详细权限标识 | 说明              |
| -------- | ---------- | ------------ | ----------------- |
| N        | 无访问权限 | DE           | 删除              |
| F        | 完全控制   | RD           | 读取数据/列出目录 |
| M        | 修改权限   | WD           | 写入数据/添加文件 |
| RX       | 读取+执行  | X            | 执行/遍历         |
| R        | 只读       | RC           | 读取控制          |
| W        | 只写       | WDAC         | 写入DAC           |

##### 示例

```batch
:: 保存C:\Windows及其子目录的ACL到文件
icacls c:\windows\* /save AclFile /T

:: 授予管理员对file文件的删除+写入DAC权限
icacls file /grant Administrator:(D,WDAC)
```

### 2.3 批处理核心命令（CALL）

#### 语法

```batch
:: 调用外部批处理文件
CALL [drive:][path]filename [batch-parameters]

:: 调用批处理内部标签
CALL:label arguments
```

#### 批处理参数扩展（命令扩展增强）

| 扩展语法 | 说明                           |
| -------- | ------------------------------ |
| %~1      | 移除%1参数的引号               |
| %~f1     | 扩展%1为完整绝对路径           |
| %~d1     | 仅扩展%1为驱动器号（如C:）     |
| %~p1     | 仅扩展%1为路径（不含驱动器）   |
| %~n1     | 仅扩展%1为文件名（不含扩展名） |
| %~x1     | 仅扩展%1为文件扩展名           |
| %~dp1    | 扩展%1为驱动器号+路径          |

### 2.4 目录操作（CD/CHDIR）

#### 语法

```batch
CHDIR [/D] [drive:][path]
CHDIR [..]
CD [/D] [drive:][path]
CD [..]
```

#### 参数说明

| 参数   | 说明                                      |
| ------ | ----------------------------------------- |
| /D     | 同时切换驱动器和目录（如`cd /d D:\test`） |
| ..     | 切换到父目录                              |
| 无参数 | 显示当前驱动器和目录                      |

#### 特性

命令扩展启用时，支持无引号的空格目录名（如`cd \winnt\profiles\username\programs\start menu`），自动匹配磁盘目录的大小写。

### 2.5 磁盘管理命令

#### 2.5.1 CHKDSK（磁盘检查与修复）

##### 语法

```batch
CHKDSK [volume[[path]filename]]] [/F] [/V] [/R] [/X] [/I] [/C] [/L[:size]] [/B] [/scan] [/spotfix]
```

##### 核心参数

| 参数     | 说明                                       |
| -------- | ------------------------------------------ |
| /F       | 修复磁盘错误（需独占访问，可能强制卸除卷） |
| /R       | 查找坏扇区并恢复可读数据（隐含/F）         |
| /X       | 强制卸除卷（所有打开句柄失效，隐含/F）     |
| /scan    | NTFS卷联机扫描（不中断系统使用）           |
| /spotfix | NTFS卷执行点修复（快速修复已知问题）       |

#### 2.5.2 CONVERT（FAT转NTFS）

##### 语法

```batch
CONVERT volume /FS:NTFS [/V] [/CvtArea:filename] [/NoSecurity] [/X]
```

##### 关键参数

| 参数        | 说明                             |
| ----------- | -------------------------------- |
| /FS:NTFS    | 强制指定转换为NTFS格式           |
| /X          | 强制卸除卷（确保转换无文件占用） |
| /NoSecurity | 转换后所有用户可访问文件/目录    |

### 2.6 文件操作命令

#### 2.6.1 COPY（文件复制/拼接）

##### 语法

```batch
COPY [/D] [/V] [/N] [/Y | /-Y] [/Z] [/L] [/A | /B ] source [/A | /B] [+ source [/A | /B] [+ ...]] [destination [/A | /B]]
```

##### 核心参数

| 参数            | 说明                                     |
| --------------- | ---------------------------------------- |
| /Y              | 覆盖目标文件不提示（批处理默认）         |
| /-Y             | 覆盖目标文件前提示（交互模式默认）       |
| /Z              | 联网文件的可重启复制（断网后恢复）       |
| source1+source2 | 多文件拼接（如`copy a.txt+b.txt c.txt`） |

#### 2.6.2 COMPACT（NTFS文件压缩）

##### 语法

```batch
COMPACT [/C | /U] [/S[:dir]] [/A] [/I] [/F] [/Q] [/EXE[:algorithm]] [/CompactOs[:option]] [filename [...]]
```

##### 核心参数

| 参数              | 说明                                             |
| ----------------- | ------------------------------------------------ |
| /C                | 压缩文件/标记目录为“自动压缩新文件”              |
| /U                | 解压缩文件/取消目录自动压缩                      |
| /EXE[:algorithm]  | 可执行文件优化压缩（算法：XPRESS4K/8K/16K、LZX） |
| /CompactOs:always | 压缩所有系统二进制文件                           |

### 2.7 系统显示与配置

#### 2.7.1 COLOR（控制台颜色设置）

##### 语法

```batch
COLOR [attr]
```

##### 颜色属性说明

`attr`为两位十六进制数（第一位=背景，第二位=前景），颜色值对照表：
| 数值 | 颜色 | 数值 | 颜色 |
|------|------|------|------|
| 0 | 黑色 | 8 | 灰色 |
| 1 | 蓝色 | 9 | 淡蓝色 |
| 2 | 绿色 | A | 淡绿色 |
| 3 | 浅绿色 | B | 淡浅绿色 |
| 4 | 红色 | C | 淡红色 |
| 5 | 紫色 | D | 淡紫色 |
| 6 | 黄色 | E | 淡黄色 |
| 7 | 白色 | F | 亮白色 |

##### 示例

```batch
:: 前景淡红、背景亮白
COLOR fc
```

#### 2.7.2 CHCP（代码页切换）

##### 语法

```batch
CHCP [nnn]
```

##### 说明

- 无参数：显示当前活动代码页；
- `nnn`：指定代码页编号（如936=简体中文、65001=UTF-8）。

| 注册表路径                                                 | 作用                      | 取值                               |
| ---------------------------------------------------------- | ------------------------- | ---------------------------------- |

## 4. 启动与系统配置命令

### 4.1 BCDEDIT（启动配置数据编辑器）

- **作用**：管理系统启动项，替代传统`Boot.ini`，用于BIOS/EFI系统启动配置
- **核心语法**
  ```batch
  BCDEDIT <命令> [参数]
  ```
- **常用命令**
  | 命令 | 作用 |
  |------|------|
  | /enum | 列出启动项（默认枚举ACTIVE项） |
  | /default | 设置默认启动项 |
  | /timeout | 设置启动菜单超时时间 |
  | /delete | 删除启动项 |
  | /set | 修改启动项参数 |

### 4.2 SHUTDOWN（系统关机/重启）

- **语法**
  ```batch
  SHUTDOWN [/i | /l | /s | /r | /a | /p | /h] [/t xxx] [/c "注释"] [/f]
  ```
- **核心参数**
  | 参数 | 作用 |
  |------|------|
  | /s | 关机 |
  | /r | 重启 |
  | /l | 注销 |
  | /a | 取消关机 |
  | /t xxx | 倒计时xxx秒后执行 |
  | /f | 强制关闭程序 |
  | /h | 休眠 |
- **示例**
  ```batch
  :: 60秒后强制重启
  SHUTDOWN /r /t 60 /f
  :: 取消关机
  SHUTDOWN /a
  ```

### 4.3 START（启动程序/命令）

- **语法**
  ```batch
  START ["标题"] [/D 路径] [/MIN | /MAX] [/B] 程序/命令 [参数]
  ```
- **核心参数**
  | 参数 | 作用 |
  |------|------|
  | /MIN | 最小化启动 |
  | /MAX | 最大化启动 |
  | /B | 不新建窗口运行 |
  | /WAIT | 等待程序结束再返回 |
- **示例**
  ```batch
  :: 最小化启动记事本
  START /MIN notepad.exe
  ```

### 4.4 SUBST（虚拟驱动器映射）

- **语法**
  ```batch
  SUBST <虚拟驱动器>: <物理路径>
  SUBST <虚拟驱动器>: /D
  ```
- **示例**
  ```batch
  :: 将D:\Test映射为Z盘
  SUBST Z: D:\Test
  :: 删除虚拟驱动器Z
  SUBST Z: /D
  ```

## 5. 文件关联与类型命令

### 5.1 ASSOC（文件扩展名关联）

- **作用**：设置/查看文件扩展名与文件类型的关联
- **语法**
  ```batch
  ASSOC [.ext[=[fileType]]]
  ```
- **示例**
  ```batch
  :: 查看.txt关联
  ASSOC .txt

  ASSOC .txt=txtfile
  ```

- **作用**：设置文件类型的默认打开程序
- **语法**
  ```batch
  FTYPE [fileType[=[打开命令]]]
  ```
- **示例**
  ```batch
  :: 设置txtfile用记事本打开
  FTYPE txtfile=notepad.exe %1
  ```

## 12. 链接与高级目录操作

### 12.1 MKLINK（创建链接）

- **语法**
  ```batch
  MKLINK [[/D | /H | /J]] 链接名 目标路径
  ```
- **参数**
  | 参数 | 作用 |
  |------|------|

## 18. 批处理循环与流程控制核心（FOR命令全解）

FOR是CMD批处理中最核心的循环命令，支持文件遍历、数字序列、文本解析、命令输出处理等场景，命令扩展启用时支持多种增强格式。

### 18.1 基础语法

```batch
FOR %变量 IN (集合) DO 命令 [参数]
```

- 批处理文件中必须使用`%%变量`，CMD交互窗口使用`%变量`
- 变量为**单个英文字母**，区分大小写（`%i`≠`%I`）
- 集合支持通配符（`*`/`?`）、多个文件/值，用空格分隔

### 18.2 增强格式（命令扩展启用）

#### 18.2.1 /D 目录遍历

- **作用**：仅匹配目录名，不匹配文件名
- **语法**
  ```batch
  FOR /D %变量 IN (目录通配符) DO 命令
  ```
- **示例**
  ```batch
  :: 遍历D盘根目录下的所有文件夹
  FOR /D %%i IN (D:\*) DO echo 找到目录：%%i
  ```

#### 18.2.2 /R 递归目录遍历

- **作用**：递归遍历指定目录树的所有子目录，执行循环
- **语法**
  ```batch
  FOR /R [[驱动器:]路径] %变量 IN (集合) DO 命令
  ```
- **示例**
  ```batch
  :: 递归遍历D:\Test下所有.txt文件
  FOR /R D:\Test %%i IN (*.txt) DO echo 找到文件：%%i
  ```

#### 18.2.3 /L 数字序列循环

- **作用**：按指定的起始值、步长、结束值生成数字序列，执行循环
- **语法**
  ```batch
  FOR /L %变量 IN (起始值,步长,结束值) DO 命令
  ```
- **示例**
  ```batch
  :: 循环输出1-10的数字
  FOR /L %%i IN (1,1,10) DO echo 数字：%%i
  :: 循环创建1-5号文件夹
  FOR /L %%i IN (1,1,5) DO MD Dir%%i
  ```

#### 18.2.4 /F 文本/命令输出解析

- **作用**：解析文本文件、字符串、命令输出的内容，按规则拆分字段，是批处理文本处理的核心
- **基础语法**
  ```batch
  FOR /F ["选项"] %变量 IN (文件集/字符串/命令) DO 命令
  ```
- **核心解析选项**
  | 选项 | 作用说明 |
  |------|----------|
  | eol=c | 指定行注释字符，以该字符开头的行将被忽略 |
  | skip=n | 跳过文件开头的n行 |
  | delims=xxx | 指定分隔符集合，替换默认的空格/制表符 |
  | tokens=x,y,m-n | 指定每行要提取的字段，支持范围、通配符\* |
  | usebackq | 启用新语法：反引号包裹执行命令、双引号包裹文件名、单引号包裹纯字符串 |
- **高频示例**
  ```batch
  :: 解析test.txt，按逗号分隔，提取第2、3列内容
  FOR /F "tokens=2,3 delims=," %%i IN (test.txt) DO echo %%i %%j
  :: 执行ipconfig命令，提取包含IPv4的行
  FOR /F "delims=" %%i IN ('ipconfig ^| findstr "IPv4"') DO echo %%i
  ```

### 18.3 FOR 变量扩展增强

和CALL的参数扩展一致，FOR变量支持以下扩展语法（以`%%i`为例）：
| 扩展语法 | 作用说明 |
|----------|----------|
| %~i | 移除变量值的引号 |
| %~fi | 扩展为完整的绝对路径 |
| %~di | 仅扩展为驱动器号 |
| %~pi | 仅扩展为路径（不含驱动器） |
| %~ni | 仅扩展为文件名（不含扩展名） |
| %~xi | 仅扩展为文件扩展名 |
| %~zi | 扩展为文件大小（字节） |
| %~ti | 扩展为文件的修改时间 |

## 19. 批处理错误处理与状态管理

### 19.1 ERRORLEVEL（退出码判断）

- **核心作用**：CMD中每个命令执行后都会返回一个数字退出码（0=成功，非0=失败/异常），通过ERRORLEVEL可判断命令执行结果

  ```batch
      echo 复制失败
  )
  ```

- **语法**
  ```batch
  :: 退出CMD窗口，设置退出码
  EXIT [退出码]

  EXIT /B [退出码]
  ```
- **示例**
  ```batch
  :: 批处理执行失败，退出并返回错误码1
  EXIT /B 1
  ```

## 22. CMD批处理最佳实践与常见避坑指南

1. **路径空格处理**：所有包含空格的路径必须用英文双引号包裹，例如`"D:\Program Files\Test"`，避免CMD解析错误
2. **延迟变量坑**：FOR循环、IF复合语句中，必须启用`SETLOCAL ENABLEDELAYEDEXPANSION`，用`!变量!`替代`%变量%`，否则变量不会实时更新
3. **管理员权限**：BCDEDIT、SC、ICACLS、FORMAT等命令必须以**管理员身份**运行CMD，否则会提示权限不足
4. **引号处理规则**：使用`CMD /C`执行带引号的多命令时，注意`/S`开关对引号的处理逻辑，避免命令执行异常
5. **批处理编码**：批处理文件建议使用ANSI编码（GB2312/GBK），避免中文乱码；如需UTF-8，需在脚本开头添加`CHCP 65001`
6. **命令优先级**：命令行开关 > 批处理SETLOCAL设置 > 注册表配置 > 系统默认值，调试时优先检查命令行参数
